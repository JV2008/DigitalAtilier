# Digital Atelier

Projeto de landing page para um estúdio de curadoria de arte digital e produtos premium.

## Descrição

Esta página apresenta uma interface elegante com foco em:

- branding refinado (`The Digital Atelier`)
- apresentação de coleção nova (`NEW COLLECTION 2024`)
- seção de produto em destaque
- cards de valores da marca
- depoimentos de clientes
- cadastro para newsletter
- rodapé com links de navegação e redes sociais

## Estrutura do projeto

- `index.html` - marcação base da página
- `style.css` - estilos de layout e responsividade
- `001-stars.png` - imagem usada nos depoimentos
- `VASO.png` - imagem de vaso para versão desktop no bloco de destaque

## Tecnologias utilizadas

- HTML5
- CSS3

## Layout

A página usa uma estrutura mobile-first e adapta o layout para telas maiores com media queries. O principal header é simples e limpo, com ícones e título central.

A seção `.home` contém:

- um badge de coleção nova
- título e subtítulo
- botões de ação (`Shop Now`, `View Lookbook`)
- imagem principal com box de texto sobreposto

## Estilização

O CSS aplica:

- variáveis CSS para cores principais
- bordas arredondadas
- efeitos de hover em botões e cards
- grid e flexbox para layout responsivo
- imagem de destaque com transformação leve

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Edite `style.css` para ajustar cores, fontes e responsividade.
3. Substitua as imagens se desejar personalizar o visual.

## Observações

O projeto já inclui ajustes de responsividade para telas maiores a partir de `720px`, onde o layout muda para colunas e o bloco de destaque se reposiciona.
